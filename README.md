# Cocktail APP

[Node.js](https://nodejs.org/en/) application built with [Express](https://expressjs.com/).

## Run Locally

Clone the project

https://github.com/Viorel1989/CocktailApp.git


Go to the project directory

```bash
  cd geolocation
```

```bash
  npm install
```

```bash
  npm run devStart
```

Open http://localhost:3000/ in browser
